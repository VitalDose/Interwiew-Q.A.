Тестирование - – процесс исследования ПО с целью получения информации о качестве продукта;
SOAP - протокол передачи данных в формате XML
Особенности: наличие WSDL, работает с различнимы протоколами, не может быть кеширован, это протокол.
XSD - схема которая описывает структуру XML документа и список даных которые в ней хранятся
WSDL - описывает структуру нашего сервиса
REST - архитектурный стиль с помощью которого описывается структура передачи информации (в стиле не используют WSDL у него меньше правил)
RESTful - веб-сервис которых отвечает требованиям REST
